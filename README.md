# Movie Review Sentiment Analysis

Multi-class sentiment classification project that predicts whether a movie review is negative, neutral, or positive using Natural Language Processing (NLP) and machine learning techniques.

## Dataset

The dataset contains movie review text (`phrase`) along with engineered text features.

**Target Variable:** `sentiment`

| Class | Label |
|---------|---------|
| 0 | Negative |
| 1 | Neutral |
| 2 | Positive |

## Methodology

- Text preprocessing and cleaning
- Exploratory data analysis
- Feature engineering
- TF-IDF vectorization
- Model training and evaluation
- Hyperparameter tuning

## Models

- Logistic Regression
- Support Vector Machine
- Naive Bayes
- Random Forest
